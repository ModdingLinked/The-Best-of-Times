Installing Tale of Two Wastelands

In this step, we will install and activateTale of Two Wastelands through Mod Organizer 2.

Installing TTW

  1. Download the 3.3.2a Installer and extract the contents of the archive anywhere outside of the default Windows folders.
  2. Open Mod Organizer 2, select the button at the top and select Open Mods folder.
  3. Here, create a folder named Tale of Two Wastelands and then copy its path.
  4. Close MO2.
  5. Back in the TTW Installer folder, right-click the TTW Install.exe and select Run As Administrator.
  6. In the installer, verify that the automatically set file paths for both your Fallout 3 and Fallout New Vegas installations are correct.
  7. Paste the path you copied in step 3 in the last text field.
  8. Ensure your paths look similar to the example above, then Install.
The installation might take a long time as the audio encoding section is very CPU-intensive. It takes about 40 minutes on a Ryzen 5 5600x but could take several hours or even longer on a slower CPU. It is recommended to close any other CPU-intensive application during this time to avoid slowdowns.

If you run into any errors while installing, check the FAQ to see if it has the solution.

Activating and Size Check

  1. Open Mod Organizer 2.
  2. If you installed TTW correctly, you should see the Tale of Two Wastelands mod in the left pane of MO2.
    • If you do not see anything or the mod name is greyed out, you did not install TTW to the correct folder. Find out where the files were installed to and move them to the mods folder.
      The path should look like this: YOUR MOD ORGANIZER FOLDER\mods\Tale of Two Wastelands
  3. Click the checkbox next to it in the left pane and the right pane should fill with plugins.
  4. In MO2, right-click Tale of Two Wastelands and Open in Explorer.
  5. Right-click anywhere in the folder (do not select any file inside) and open its Properties.
  6. If the size of your folder does not match 15.7GB, reinstall the Tale of Two Wastelands. This can be done by first clearing the contents of the current folder, then repeating the installation steps.
