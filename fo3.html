IMPORTANT:
This spin-off guide is intended for people who want to play Fallout 3 but without Tale of Two Wastelands. For the sake of brevity, instructions are not as fleshed out as in the main guides, requiring more familiarity with modding.

Short Intruduction

Requirements

The information from TBoT's Introduction and Setup pages doesn't change, you obviously can ignore the parts about Fallout: New Vegas.

Once you have a clean slate, create a new MO2 instance for Fallout 3 and start installing the mods as instructed.

Support for this guide will be limited! You can use the dedicated thread in our Discord server.

Manual Downloads

Root Mods

The following mods are special because they need to be installed directly into the game's Root folder, not through Mod Organizer 2.
Use the Manual Download and follow the instructions provided for each mod.

Failing to install these mods correctly will result in issues.

Extends the scripting capabilities of the game.

Installation instructions

  • Main Files - Fallout Script Extender (FOSE)
    1. From the downloaded archive, move everything to the game's Root folder.

    If you do not know what the Root folder is, read the Key Terminology.

FO3 BSA Decompressor

Decompresses the vanilla BSA files to reduce loading times and stuttering. Can also fix certain sound effects not playing.

Installation instructions

  • Main Files - FO3 BSA Decompressor
    1. Extract the archive and run FO3 BSA Decompressor.exe.
    2. The Fallout 3 and Decompressed Archives paths should be filled by default (Root folder and Data folder respectively). If they aren't, close the program and re-run your game launcher to generate the required registry key.

      3. The EGS release of the game does not create any keys! You need to select the game folder manually!

    3. Click Decompress, wait for the process the finish, then exit the program once finished.

Unofficial Fallout 3 ESM Patcher

Optimizes and fixes thousands of records across the base game and DLC plugins to increase performance and stability.

Installation instructions

  • Main Files - Unofficial Fallout 3 ESM Patcher
    1. Click the button next to the profiles bar in MO2 and select Open Mods folder.
    2. Create an empty folder called Fixed ESMs inside of the Mods folder.
    3. Extract the archive and run Unofficial Fallout 3 ESM Patcher.exe.
    4. The Fallout 3 path should be filled by default (Root folder). If it isn't, close the program and re-run your game launcher to generate the required registry key.

      5. The EGS release of the game does not create any keys! You need to select the game folder manually!

    5. Click Browse next to the Patched Files path field, then choose the Fixed ESMs folder you just created.
    6. Click Install, wait for the process the finish, then exit the program once finished.
    7. In Mod Organizer 2, press F5 to refresh the left pane and enable the Fixed ESMs mod by ticking its checkbox.
    8. You can now delete files created in the step 1.

Game Configuration

Custom INI

In this step we will take advantage of Command Extender's FalloutCustom.ini feature to improve performance and stability without affecting the main INIs.

FalloutCustom.ini is a special settings file that overrides all other game INIs. This INI is exclusive to the user, meaning that the game or its launcher cannot change it.

This allows you to change settings without the fear of being overwritten, and removes the need of tracking which settings go into which file.

  1. Click the button at the top of MO2 and select INI Editor.
  2. Select the FalloutCustom.ini tab, which should be blank. In case it's not, clear the file.

Performance Guide

In-depth guide on how to fix the game's lackluster performance, display issues and alt-tabbing. Covers topics such as DXVK, framerate limiting, display modes, lag and HDR.

Most of the information in the Performance Guide applies to Fallout 3 as well, here's a list of the instructions that differ:

  • Ignore the Engine Optimizations section, lStewieAl's Tweaks is already part of the list.
  • The equivalent of NVTF's bUseDefaultPoolForTextures is bDontMirrorTexturesInRAM from lStewieAl's Tweaks. Triple buffering is enabled by Triple Buffering (FOSE).
  • Ignore the Fog Culling section, the mod does not exist for FO3.
  • Ignore the zlib section, zlib Updated is already part of the list.
  • If you are going to use HDR, know that the HDR Save Image Patch mod does not exist for FO3.
  • Use this version of OneTweak.
